1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. Constants should only be used when the value is not meant to change during the programs execution. Variables should be used when the value is expected to be changed later in the programs execution. If all values were constants, the program would be very limited in its functionality.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parentheses affect the arithmetic outcome by changing the order of operations. Operations within parentheses are performed first, followed by multiplication and division, and finally addition and subtraction.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. We can limit floating-point numbers to a given number of decimals by using the toFixed() method, which formats a number using fixed-point notation.

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. We can compare two floating-point values by checking if the absolute difference between them is smaller than a very small number (epsilon). This accounts for potential precision errors in floating-point arithmetic.

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Comparison operators return a boolean value, either TRUE or FALSE, based on the comparison of two values. We can check if they are FALSE or TRUE by using conditional statements like if-else or by directly logging the result to the console.

  6. How does the Math.random() function work, and what can it be used for?

    7. Math.random() creates a random floating-point number between 0 and 1. It can be used to generate a random value which can later be manipulated and used in functions and calculations.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. Modulus can be used to find the remainder of a division operation. It is useful in various scenarios, such as determining if a number is even or odd, cycling through a set of values, or constraining a value within a specific range.
